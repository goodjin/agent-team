# 🎮 字母大挑战 - 网页版字母小游戏

## 游戏简介

"字母大挑战"是一款简单有趣的网页版字母识别游戏，旨在帮助玩家提高字母识别速度和键盘输入能力。

## 游戏特色

### ✨ 核心功能
- **随机字母显示**：屏幕随机显示A-Z的大写字母
- **快速响应**：输入正确后立即跳转到下一个字母
- **实时反馈**：正确/错误都有视觉反馈效果
- **倒计时系统**：限时挑战，增加游戏紧张感

### 🎯 游戏机制
- **连击系统**：连续答对获得连击加成分数
- **计分系统**：
  - 答对：+10分（每3连击额外+5分）
  - 答错：-5分
- **三种难度**：
  - 简单：30秒
  - 中等：20秒
  - 困难：10秒

### 📊 游戏统计
游戏结束后显示详细统计数据：
- 最终得分
- 正确答案数量
- 错误答案数量
- 最高连击数
- 平均答题速度（个/秒）

## 如何使用

### 方式一：直接打开
1. 双击 `letter_game.html` 文件
2. 在浏览器中即可开始游戏

### 方式二：本地服务器（推荐）
```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 然后在浏览器访问
# http://localhost:8000/letter_game.html
```

## 游戏操作

### 开始游戏
1. 选择难度等级（简单/中等/困难）
2. 点击"开始游戏"按钮

### 游戏中
- 观察屏幕上显示的字母
- 在输入框中输入对应的字母
- 输入正确后自动跳到下一个字母
- 可以使用回车键快速提交

### 游戏结束
- 时间到或点击"结束游戏"
- 查看统计数据
- 选择"再来一局"或"返回菜单"
